Arquitectura

Diseño de 3 procesos: motor Rust (Orion), UI React y backend Node.js. IPC, WebView2 y pipeline de compilación.

Rust React Node.js

Seguridad y Privacidad

HTTPS-only, HSTS preload, bloqueo de trackers (EasyList), CSP, permisos por sitio y cero telemetría.

Activo

Inicio Rápido

Listo en 3 pasos

Sin instaladores complejos. Sin configuración extra.

1

Descarga el .exe

Un solo archivo ejecutable. ~45 MB. No requiere instalador.

2

Ejecuta

Doble clic. Windows puede mostrar una alerta SmartScreen — clic en "Más información" → "Ejecutar de todos modos".

3

Navega

Flux inicia con tu página de bienvenida. Todos tus datos se guardan localmente en flux.db.
