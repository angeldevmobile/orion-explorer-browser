Guía de Instalación

Primeros Pasos

Descarga, instala y lanza Flux Browser en menos de 60 segundos. Sin instaladores complejos ni dependencias externas.

v1.0.0-beta.1 Windows 10 / 11

Requisitos del sistema

Componente Mínimo Recomendado
Sistema operativo Windows 10 (build 1803+) Windows 11
RAM 4 GB 8 GB o más
Almacenamiento 150 MB libres 500 MB+ (para caché y flux.db)
WebView2 Runtime Requerido Preinstalado en Windows 11
Conexión a internet Requerida para navegar —

ℹ WebView2 Runtime está incluido por defecto en Windows 11. En Windows 10 se instala automáticamente junto con las actualizaciones de Microsoft Edge, o puedes descargarlo desde microsoft.com.

Instalación en 3 pasos

1

Descarga el ejecutable

El navegador se distribuye como un único archivo .exe de ~45 MB. No hay instalador. No hay proceso de setup.

Descargar Flux Browser
2

Ejecuta el archivo

Haz doble clic sobre flux-browser.exe. La primera vez, Windows puede mostrar una alerta SmartScreen porque el ejecutable no está firmado digitalmente aún.

Alerta SmartScreen de Windows

Haz clic en "Más información" y luego en "Ejecutar de todos modos". Flux no tiene malware — la alerta aparece porque la firma de código está en proceso para la beta.

3

Navega

Flux inicia directamente con tu página de bienvenida. Todos tus datos (historial, favoritos, preferencias) se guardan automáticamente en flux.db — una base de datos SQLite local en la misma carpeta del ejecutable.

Qué viene en la descarga

flux-browser/
  flux-browser.exe     ← abrir esto
  flux-backend.exe    ← arranca automáticamente
  yt-dlp.exe         ← bundleado para descargar video/audio
  flux.db            ← se crea en el primer arranque

No se requiere: Node.js · PostgreSQL · Docker · Python · yt-dlp manual.

Configuración inicial (opcional)

Flux AI con Gemini

Para usar el asistente IA, el traductor y la detección de canciones necesitas una API key de Google Gemini (gratuita). Añádela en flux://settings → IA.

Búsqueda privada con SearXNG

Si quieres búsqueda 100% privada sin DuckDuckGo, puedes lanzar un SearXNG local con Docker. Sin él, la búsqueda usa DuckDuckGo automáticamente.

docker compose up -d

Crear una cuenta

La cuenta de usuario es opcional y 100% local. Te permite guardar historial personalizado, favoritos sincronizados entre sesiones y re-ranking de búsqueda personalizado.

Tema visual

Flux incluye selector de temas en flux://settings → Apariencia. El tema oscuro con acento cyan es el predeterminado.
