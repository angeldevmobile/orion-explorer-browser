v1.0.0-beta.1

Beta pública Marzo 2026

Primera versión pública de Flux Browser. Motor Rust operativo, UI React completa, backend standalone y todas las funcionalidades core disponibles.

flux-engine Motor Rust

  • ✓ Tokenizer HTML zero-copy basado en &str — sin allocations del HTML fuente
  • ✓ Parser DOM arena-based con Vec<Node> + NodeId — cache-friendly, sin Box/Rc
  • ✓ CSS cascade con especificidad completa (tag · clase · id · inline) y UA stylesheet
  • ✓ Inline layout + word wrapping + text-align (left/center/right)
  • ✓ FluxSoftRenderer — pixel buffer XRGB8888 con fontdue glyph atlas y alpha blending
  • ✓ JavaScript con QuickJS (rquickjs) — DOM bindings completos, fetch() real, eventos, sandbox
  • ✓ SecurityLayer: HTTPS-only, HSTS preload, CSP enforcement, ad/tracker blocker
  • ✓ Bloqueador de anuncios YouTube — 3 capas: scriptlet + CSS cosmético + intercepción fetch/XHR
  • ✓ Descargador de medios con yt-dlp bundleado y progreso en tiempo real via IPC
  • ✓ Mute por pestaña via IPC
  • ✓ Sistema de permisos nativos (cámara, micrófono, notificaciones) implementado en Rust
  • ✓ Manejo graceful de fallos del engine — si el puerto 4000 está ocupado el browser sigue

wry / tao Ventana nativa

  • ✓ Ventana nativa con tao — sin decoraciones, transparente, frameless
  • ✓ chrome_view (React UI) + content_view (páginas externas) en dos WebViews separados
  • ✓ IPC bidireccional completo entre React y Rust
  • ✓ Resize dinámico de los dos WebViews según la altura del chrome
  • ✓ HTTPS upgrade antes de load_url()
  • ✓ flux-backend.exe como sidecar — arranca y cierra automáticamente con el browser

Flux Search Motor de búsqueda

  • ✓ SearXNG self-hosted (Docker, opcional) — búsqueda privada sin Google/Bing
  • ✓ Fallback automático a DuckDuckGo si SearXNG no está activo
  • ✓ Re-ranking BM25 en Rust (flux-engine :4000 POST /process)
  • ✓ Ranking personalizado por historial — boost proporcional a frecuencia de visitas
  • ✓ Caché de búsquedas en memoria con TTL de 5 minutos
  • ✓ Autocompletado en barra de URL con historial y favoritos
  • ✓ Resumen IA de resultados via Gemini (endpoint separado, no bloquea la carga)
  • ✓ Paginación de resultados

flux-backend Backend y persistencia

  • ✓ Migración a SQLite — sin PostgreSQL, sin Docker, cero dependencias para el usuario
  • ✓ Distribución como exe standalone — compilado con @yao-pkg/pkg, no requiere Node.js
  • ✓ Auth JWT + bcrypt — cuentas de usuario 100% locales
  • ✓ Historial, favoritos, tabs, grupos de tabs, preferencias persistentes
  • ✓ Estadísticas de navegación — BrowsingStats, SiteVisit, HourlyActivity
  • ✓ Gemini IA — chat, resúmenes, OCR, detección de canciones
  • ✓ Servicio de traducción — Gemini + fallback MyMemory (15 idiomas, sin API key extra)
  • ✓ Noticias RSS, clima (wttr.in), tendencias (Google Trends)
  • ✓ Registro de descargas de medios

UI Interfaz React

  • ✓ Chrome React con identidad visual Flux (dark, cyan, glassmorphism)
  • ✓ Barra de direcciones con búsqueda integrada y autocompletado
  • ✓ Sistema de tabs con grupos y colores personalizados
  • ✓ Panel lateral flotante de IA (FluxAISidePanel) — contextual con URL y título actual
  • ✓ Nueva pestaña con clima, noticias RSS, tendencias y búsqueda por voz
  • ✓ Descargador de medios con progreso en tiempo real via IPC
  • ✓ Focus Mode con lista de sitios bloqueados y sesiones de enfoque
  • ✓ Reader Mode, Split view, Panel de favoritos
  • ✓ Modo privado / incógnito por pestaña
  • ✓ Onboarding de bienvenida al primer arranque
  • ✓ OCR (Gemini Vision) y Detección de canciones (Gemini Audio)
  • ✓ Selector de tema, estadísticas de uso, dev tools, view-source
  • ✓ Sistema de login y registro de usuario con auth local
