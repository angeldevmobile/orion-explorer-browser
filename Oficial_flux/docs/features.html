Seguridad y Privacidad Activo

Flux está diseñado desde cero con un modelo privacy-first y local-first. Ningún dato del usuario sale del dispositivo sin consentimiento explícito.

Capa de protecciónQué hace
HTTPS-onlyUpgrade automático HTTP → HTTPS en todas las peticiones antes de cargar la página
HSTS preloadLista embebida de 20+ dominios de alto tráfico que siempre usan HTTPS — sin primera petición insegura
CSP enforcementParseo y aplicación del Content-Security-Policy del servidor. Bloquea scripts inline no autorizados y fetch() fuera de connect-src
JS sandboxQuickJS con 16 MB de heap y 512 KB de stack máximo — sin acceso al filesystem ni al SO
Permisos explícitosCámara, micrófono y geolocalización requieren confirmación del usuario via barra de permisos nativa
Sin telemetríaFlux no envía datos de uso, crashlogs ni métricas a ningún servidor externo
Rust sin memory bugsBuffer overflows, use-after-free y null pointers son imposibles en safe Rust

Lo que Flux NO hace (a diferencia de Chrome): Sin cuenta obligatoria · Sin sincronización sin que lo actives · Sin historial de búsquedas en servidores externos · Sin fingerprinting de hardware · Sin extensiones de terceros con acceso a tus datos.

Bloqueador de Anuncios Estable

Bloqueo integrado en el engine Rust. Sin extensiones externas. Sin overhead de proceso separado.

Capa 1 — Network

Lista de ~40 dominios de rastreo y publicidad (EasyList/EasyPrivacy subset) integrada en el engine Rust. Las peticiones se bloquean antes de salir al network.

Capa 2 — YouTube

Parchea ytInitialPlayerResponse para eliminar adPlacements. CSS cosmético oculta overlays residuales.

Capa 3 — Intercepción

Intercepta fetch() y XHR para bloquear URLs de tracking como /api/stats/ads y /pagead/.

El panel de seguridad (icono de escudo en la barra) muestra cuántos trackers fueron bloqueados en la página actual.

Flux Search Estable

Motor de búsqueda propio. Sin Google. Sin Bing. Resultados re-rankeados con BM25 propio y personalizados por tu historial de navegación.

Flujo de búsqueda:

Usuario escribe → flux://search?q=...
  ↓ flux-backend (:3000)
    SearXNG self-hosted (:8080) si está disponible
    DuckDuckGo fallback automático
  ↓ flux-engine (:4000)
    BM25 re-ranking + boost por historial personal
  ↓ SearchPage React — resultados con identidad Flux

Autocompletado inteligente

La barra de URL sugiere en tiempo real: historial personal, favoritos, búsquedas anteriores y URLs visitadas.

Caché de búsquedas

TTL de 5 minutos en memoria. Repetir la misma búsqueda no golpea SearXNG ni el engine — respuesta instantánea.

Resumen IA

Endpoint separado con Gemini que genera un resumen de los resultados sin bloquear la carga principal. Requiere API key.

Ranking personalizado

Los sitios que visitas frecuentemente suben en los resultados de búsqueda — boost proporcional a frecuencia de visitas.

Flux AI Estable Requiere Gemini API key

Asistente IA integrado en el browser, consciente del contexto de navegación actual.

Panel lateral flotante

Abre un chat Gemini contextual que sabe qué página estás visitando. Puedes preguntarle sobre el contenido sin copiar y pegar. Renderizado Markdown con animación de escritura.

Página completa flux://ai

Interfaz de chat completa para conversaciones largas. Historial de conversaciones guardado en flux.db.

Traductor integrado

Traducción de texto a 15 idiomas via Gemini con fallback a MyMemory. Sin necesidad de otra API key o extensión externa.

OCR — Texto desde imágenes

Extrae texto de cualquier imagen en la página usando Gemini Vision. Útil para capturas de pantalla, diagramas o imágenes con texto.

Detección de canciones

Detecta qué canción está sonando en la pestaña activa usando Gemini Audio. Guarda las detecciones en el historial.

Resumen de búsquedas

Genera automáticamente un resumen de los resultados de búsqueda para que no tengas que abrir cada enlace.

Descargador de Medios Estable

Descarga video y audio de cualquier página usando yt-dlp bundleado — sin instalación manual, sin extensiones externas.

Progreso en tiempo real

Porcentaje, velocidad de descarga y bytes descargados actualizados en tiempo real via IPC.

Formatos disponibles

MP4, WebM, MP3, AAC, Opus y más. Selecciona la calidad antes de descargar.

Historial de descargas

Todas las descargas quedan registradas en flux.db y accesibles desde flux://downloads.

yt-dlp.exe viene bundleado en la distribución de Flux — el usuario no necesita instalarlo por separado ni tener Python.

Modo Enfoque Estable

Bloquea sitios distractores durante sesiones de trabajo. El modo enfoque persiste entre reinicios y registra el tiempo de sesión.

Lista de sitios bloqueados

Añade cualquier dominio a la lista de bloqueo. Cuando intentas acceder durante una sesión activa, Flux te muestra una pantalla de bloqueo con el tiempo restante.

Sesiones de enfoque

Las sesiones se registran en FocusSession en flux.db. Las estadísticas de uso muestran cuánto tiempo llevas enfocado.

Reader Mode

Modo lectura que elimina la UI del browser y muestra solo el contenido de la página en una vista limpia y minimalista.

Mute por pestaña

Silencia el audio de cualquier pestaña individualmente via IPC — sin necesidad de bajar el volumen del sistema.

Nueva Pestaña Estable

La nueva pestaña de Flux es un dashboard de productividad, no una página en blanco.

Clima

Widget en tiempo real via wttr.in

BBC, TechCrunch, NASA, Al Jazeera

Tendencias

Google Trends via RSS
